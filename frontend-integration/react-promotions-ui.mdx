## Installation

Install the package using npm:

```bash
npm install @paritydeals/react-promotions-ui
```

## Getting Started

To start using the SDK, wrap your app with the `ParityDealsProvider` at the root level. This sets up the necessary context.

```tsx
import { ParityDealsProvider } from '@paritydeals/react-promotions-ui';

function App() {
  return (
    <ParityDealsProvider productId="YOUR_PRODUCT_ID">
      {/* Your app components */}
    </ParityDealsProvider>
  );
}
```

## Components

Use these ready-made UI components to easily show location-aware pricing and promotional content.

### `PDBanner`

Displays a location-aware discount banner. You can style and position it as needed.

```tsx
import { PDBanner } from '@paritydeals/react-promotions-ui';

<PDBanner
  backgroundColor="#f0f0f0"
  fontColor="#333"
  borderRadius="4px"
  fontSize="16px"
  unStyled={false}
  addCloseIcon={true}
  className="custom-banner"
  onClose={() => console.log('Banner closed')}
/>
```

---

### `PDPriceFormatted`

Simplified component to display discounted and original prices dynamically based on the user’s location.

#### Display Discounted Price

Use this to show the localized or discounted price:

```tsx
<PDPriceFormatted
  originalPrice={99.99}
  convertToLocal={true}
  showDecimal={true}
  showCurrencyCode={true}
  showCurrencySymbol={true}
/>
```

#### Display Original Price (Strikethrough)

Optionally show the original price with a strikethrough effect:

```tsx
<PDPriceFormatted
  originalPrice={99.99}
  displayPrice={99.99}
  isOriginalDisplay={true}
  convertToLocal={true}
  showCurrencySymbol={true}
/>
```

#### Example Usage

Combine both for a side-by-side comparison:

```tsx
<div className="price-container">
  <PDPriceFormatted originalPrice={89.99} showCurrencySymbol={true} />
  <PDPriceFormatted
    originalPrice={99.99}
    displayPrice={99.99}
    isOriginalDisplay={true}
    showCurrencySymbol={true}
  />
</div>
```

## Granular Price Components

Use these components individually to build custom price formats with full styling control.

### `PDPriceInteger`

Displays just the integer part of the price.

```tsx
<PDPriceInteger originalPrice={99.99} className="price-integer" />
```

### `PDPriceDecimal`

Displays just the decimal portion.

```tsx
<PDPriceDecimal originalPrice={99.99} className="price-decimal" />
```

### `PDPriceCurrencySymbol`

Renders the currency symbol (like \$, ₹, €).

```tsx
<PDPriceCurrencySymbol convertToLocal={true} className="currency-symbol" />
```

### `PDPriceCurrencyCode`

Renders the full currency code (like USD, INR, EUR).

```tsx
<PDPriceCurrencyCode convertToLocal={true} className="currency-code" />
```

### Custom Display Example

Example using all components to create a custom layout:

```tsx
<div className="custom-price-display">
  <PDPriceCurrencySymbol convertToLocal={true} />
  <PDPriceInteger originalPrice={99.99} />
  <span className="decimal-separator">.</span>
  <PDPriceDecimal originalPrice={99.99} />
  <PDPriceCurrencyCode convertToLocal={true} />
</div>
```

## Price Formatting Options

All price components accept common formatting options to customize appearance and behavior.

```ts
interface DefaultFormatConfig {
  thousandsSeparator?: string;     // default: ","
  decimalSeparator?: string;       // default: "."
  decimalPlaces?: number;          // default: 2
  roundTo?: 'none' | 'up' | 'down';
  showDecimal?: boolean;           // default: true
  convertToLocal?: boolean;        // default: false
  showCurrencyCode?: boolean;      // default: false
  showCurrencySymbol?: boolean;    // default: true
  isOriginalDisplay?: boolean;     // default: false
}
```

Use these props to align with your regional or branding standards.
